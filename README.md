## Project Description:
Battleship is a **Two-player Python terminal-based strategy game**.  
Each player secretly places a ship on a grid , and both players take turn guessing the coordinates of the opponent's ship.  
First player who correctly guesses and destroys opponents ship **Wins The Game**.

## Features:
- Two-players interactive gameplay.
- Secret ship placement for both players
- Turn-by-turn guessing system
- Simple terminal-based interface
- Instant win detection when a ship hit
- No external libraries required (pure Python)

## How to run a Game:
Make sure Python3 is installed in your system  
**Run the following command**:(on terminal)   
python3 main.py

## Game Instructions:
1. The  game starts by asking player 1 and Player 2 to place their ships secretly.
2. Each player chooses a coordinate (eg.,'A1A2') to hide their ship.
3. Player then take turns guessing the opponent's ship location.
4. If guess is correct then particular positon gets hit.
5. If like that opponent hit 1 ship completely , then game ends .

## Setup instructions :
**No External Libraries Are Required.**  
- Make sure the code is run in terminal.
